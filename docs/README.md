# docs/

FrogTailGameServer 설계 문서 저장소.

## 폴더 구조

```
docs/
├── ADR/        ← CTO 기술 결정 기록 (Architecture Decision Records)
└── design/     ← 시니어 개발자 설계 문서
```

## ADR (Architecture Decision Records)

> 작성자: CTO 에이전트
> 시점: 기술 방향 결정 / 타당성 검토 / 프로세스 결정 완료 후

- 파일명: `ADR-YYYYMMDD-{slug}.md`
- ADR은 **수정하지 않는다** — 번복 시 새 ADR + 이전 ADR 상태를 `폐기`로 변경
- 누적 기록이므로 삭제하지 않는다

## design/ (설계 문서)

> 작성자: 시니어 개발자 에이전트
> 시점: 아키텍처 설계 / 공통 모듈 설계 / 기술 병목 해결 완료 후

- 파일명: `YYYYMMDD-{slug}.md`
